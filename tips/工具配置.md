```plain
Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\*\shell\VS]

[HKEY_CLASSES_ROOT\*\shell\VS\Command]
@="D:\\Program Files\\VSCode\\Code.exe \"%1\""
```
