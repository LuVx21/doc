# 美化输出

```bash
echo '{"foo": 42, "bar": "less interesting data"}' | jq '.foo'

echo '[{"score":50,"name":"AA"}, {"score":99, "name":"BB"}]' | jq '.[]| if .score > 60 then "及格的人: "+.name else empty end'
```
