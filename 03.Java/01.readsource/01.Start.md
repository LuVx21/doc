# 阅读源码的正确姿势

insight.io

网站提供jdk源码库,也提供源码的测试例.
这个网站提供快速阅读github源码的插件,

VSCode
```
ext install Java Extendsion Pack
```
安装插件之后,在ava代码中通过`ctrl+左键`即可跳转查看源码

> 插件还支持格式化代码,Debug,单元测试,Maven项目结构等


常见IDE中

通过`ctrl+左键`即可查看源码
