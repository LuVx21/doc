# 优化

OS 的诊断主要关注的是 `CPU, Memory, I/O`

对于 CPU 主要关注平均负载(Load Average), CPU 使用率, 上下文切换次数(Context Switch).

通过 top 命令可以查看系统平均负载和 CPU 使用率,
